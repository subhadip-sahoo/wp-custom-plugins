== Description ==

Realtors plugin allowes you to search all the realtors based on the filter and get the details about them.

Major features in Realtors include:

* Get the shortcodes of form and results and use this anywhere into pages.
* It makes an api call to handle the data based on the filter options.

== Installation ==

Put the unzip plugin into plugins directory under wp-contents and activate this plugin from the wp admin. Done! Simple, isn't it?
